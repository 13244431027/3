# 参数详细分析


## 🎯 **关键参数配置表**

| 参数 | 推荐值 | 单位 | 说明 |
|------|--------|------|------|
| `miniPosX` | 0.05-0.8 | 归一化 | 距离左边距 |
| `miniPosY` | 0.05-0.8 | 归一化 | 距离底部距 |
| `w` | 0.2-0.4 | 归一化 | 相对屏幕宽度 |
| `h` | 0.25-0.45 | 归一化 | 相对屏幕高度 |
| `padding` | 5-20 | 像素 | 交互扩展区域 |
| `zoomFactor` | 1.5-3.0 | 倍率 | 视图缩放级别 |
| `camX`, `camY` | 动态 | 世界坐标 | 实时更新 |
| `bounds0-3` | 固定 | 世界坐标 | 游戏地图范围 |



## 📊 **Uniform 参数总览**

### **1. 小地图位置与尺寸参数**
```glsl
uniform float miniPosX;    // 小地图左下角X坐标 (0-1)
uniform float miniPosY;    // 小地图左下角Y坐标 (0-1)
uniform float w;           // 小地图宽度 (0-1)
uniform float h;           // 小地图高度 (0-1)
uniform float padding;     // 检测区域扩展边距 (像素)
```

**说明**：
- 使用**标准化坐标**系统：整个屏幕为 [0,1]×[0,1]
- 左下角为(0,0)，右上角为(1,1)
- `padding` 用于扩展小地图的交互检测区域，单位是像素（需要转换为标准化坐标）

### **2. 视图控制参数**
```glsl
uniform float zoomFactor;  // 缩放因子 (推荐 >1.0)
```

**值域分析**：
- `zoomFactor = 1.0`：无缩放
- `zoomFactor > 1.0`：放大（视野范围变小）
- `zoomFactor < 1.0`：缩小（视野范围变大）
- **特殊值**：
  - `zoomFactor = 2.0`：视野缩小一半
  - `zoomFactor = 0.5`：视野扩大一倍

### **3. 相机/玩家位置**
```glsl
uniform float camX;  // 相机X坐标（游戏世界坐标）
uniform float camY;  // 相机Y坐标（游戏世界坐标）
```

**坐标系转换**：
```
游戏世界坐标 → 地图标准化坐标 → 小地图UV坐标
```

### **4. 地图边界参数**
```glsl
uniform float bounds0;  // 上边界 (Y最大值)
uniform float bounds1;  // 下边界 (Y最小值)
uniform float bounds2;  // 左边界 (X最小值)
uniform float bounds3;  // 右边界 (X最大值)
```

**边界定义示例**：
```
假设地图从 (-100, -50) 到 (200, 150)
bounds0 = 150   // 上边界 (max Y)
bounds1 = -50   // 下边界 (min Y)
bounds2 = -100  // 左边界 (min X)
bounds3 = 200   // 右边界 (max X)
```

### **5. 控制开关**
```glsl
uniform bool byp;  // 绕过标志 (true=禁用效果)
```

## 🔍 **参数计算逻辑**

### **A. 地图尺寸计算**
```glsl
vec2 mapSize = vec2(
    bounds3 - bounds2,  // 地图宽度 (X方向)
    bounds0 - bounds1   // 地图高度 (Y方向)
);
```

### **B. 小地图显示尺寸计算**
```glsl
vec2 minimapSize = vec2(
    displaySize.x,  // 使用指定的宽度
    mapSize.y / mapSize.x / 3.6 * 6.4 * displaySize.x
);
```

**比例因子分析**：
- `6.4/3.6 ≈ 1.777`：这是640×360的标准宽高比
- 确保小地图**保持地图的原始宽高比**

### **C. 标准化坐标转换**

#### **像素坐标 ↔ 标准化坐标**
```glsl
// 像素坐标转标准化坐标
float normalizedX = pixelX / 640.0;
float normalizedY = pixelY / 360.0;

// 标准化坐标转像素坐标
float pixelX = normalizedX * 640.0;
float pixelY = normalizedY * 360.0;
```

#### **地图坐标 ↔ 小地图坐标**
```glsl
// 地图坐标 → 小地图坐标 (在 mapToMiniMap 函数中)
vec2 normalizedMapPos = (worldPos - vec2(bounds2, bounds1)) / mapSize;
vec2 minimapPos = normalizedMapPos * minimapSize;
```

## 📐 **参数依赖关系**

### **1. 强依赖关系**
```
camX, camY 必须在地图边界内：
    bounds2 ≤ camX ≤ bounds3
    bounds1 ≤ camY ≤ bounds0
    
否则小地图显示异常
```

### **2. 尺寸约束关系**
```
小地图显示区域不能超出屏幕：
    0 ≤ miniPosX ≤ 1
    0 ≤ miniPosY ≤ 1
    miniPosX + w ≤ 1
    miniPosY + h ≤ 1
```

### **3. 缩放因子约束**
```
zoomFactor > 0  // 必须为正数
建议范围：0.25 ≤ zoomFactor ≤ 4.0
```

## ⚠️ **边界条件与异常处理**

### **A. 危险参数组合**
```javascript
// 1. 地图尺寸为零（除零错误）
if (bounds3 == bounds2 || bounds0 == bounds1) {
    // 需要处理：mapSize = vec2(1.0, 1.0)
}

// 2. 缩放因子为零或负值
if (zoomFactor <= 0.0) {
    zoomFactor = 1.0;  // 重置为默认值
}

// 3. 小地图尺寸为零
if (w == 0.0 || h == 0.0) {
    // 小地图不可见
}
```

### **B. 相机位置约束**
```glsl
// 相机在小地图中的位置限制
vec2 minCam = displaySize * 0.5 / zoomFactor;
vec2 maxCam = minimapSize - minCam;

// 确保相机不会超出小地图边界
cam.x = clamp(player.x, minCam.x, maxCam.x);
cam.y = clamp(player.y, minCam.y, maxCam.y);
```

## 🎮 **使用场景参数配置**

### **场景1：全屏小地图**
```javascript
// 小地图覆盖整个屏幕（调试用）
miniPosX = 0.0
miniPosY = 0.0
w = 1.0
h = 1.0
zoomFactor = 1.0
```

### **场景2：右下角小地图**
```javascript
// 经典位置：屏幕右下角
miniPosX = 0.7  // 距离右边30%
miniPosY = 0.1  // 距离底部10%
w = 0.25        // 占屏幕宽度25%
h = 0.3         // 占屏幕高度30%
```

### **场景3：跟随式小地图**
```javascript
// 小地图位置随相机移动
miniPosX = clamp(camX, 0.0, 1.0-w)
miniPosY = clamp(camY, 0.0, 1.0-h)
```

## 🔧 **调试参数**

### **A. 可视化参数影响**
```glsl
// 添加调试可视化
if (debugMode == 1) {
    // 显示小地图边界
    float border = 0.005;
    bool isBorder = abs(vUv.x - miniPosX) < border || 
                    abs(vUv.x - (miniPosX + w)) < border ||
                    abs(vUv.y - miniPosY) < border || 
                    abs(vUv.y - (miniPosY + h)) < border;
    
    if (isBorder) {
        fragColor = vec4(1.0, 0.0, 0.0, 1.0);
    }
}
```

### **B. 参数范围监控**
```glsl
// 检查参数是否在有效范围内
bool validParams = true;
validParams = validParams && (zoomFactor > 0.0);
validParams = validParams && (w > 0.0 && h > 0.0);
validParams = validParams && (mapSize.x > 0.0 && mapSize.y > 0.0);

if (!validParams) {
    fragColor = vec4(1.0, 0.0, 0.0, 0.5); // 半透明红色错误提示
}
```
