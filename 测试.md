## 使用方法

1. 确保已在 **GitHub 面板 Pro+** 中加载了 `markdown-render-plus` 插件（可通过“插件管理”或“插件集市”安装）。
2. 在“浏览”模式下进入一个仓库，打开任意 `.md` 文件（或粘贴本测试内容到 AI 对话中预览）。
3. 观察页面：
   - 代码块是否高亮
   - 数学公式是否渲染
   - Mermaid 流程图是否显示为 SVG
   - Scratch 积木是否以彩色块呈现
   - 目录是否出现在右侧（或顶部），并可折叠/最小化
   - 相对图片/链接是否被自动补全为 GitHub 在线链接
