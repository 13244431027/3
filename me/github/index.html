插件合并器 v3 - 终极增强版

上传多个 GitHub 面板插件 .js 文件，或手动粘贴源码，合并为一个插件。 超级增强功能：合并后的插件会自动嵌入 可拖动、功能完整的子插件管理面板，支持在运行时对每个子插件进行启用/禁用操作。
合并后每个子插件会在独立作用域运行，并自动聚合 init、 onHook 和 style。
本工具会自动读取每个插件的 id、 name、version、 author、description。
