插件合并器（自动读取元信息）

上传多个 GitHub 面板插件 .js 文件，或手动粘贴源码，合并为一个插件。 合并后每个子插件会在独立作用域运行，并自动聚合 init、 onHook 和 style。
本工具会自动读取每个插件的 id、 name、version、 author、description。
✨ 新增功能：生成的合并插件包含可拖动控制面板，支持动态启用/禁用各子插件功能！

注意：本工具不会执行你上传的插件代码，只会读取文本并生成合并代码。 但合并后的插件加载到面板后，仍然拥有原插件的全部权限。请勿加载不可信插件。

新功能：生成的合并插件包含一个可拖动的右下角控制面板，加载后可动态启用/禁用各子插件的 init、onHook 和 style 功能。
