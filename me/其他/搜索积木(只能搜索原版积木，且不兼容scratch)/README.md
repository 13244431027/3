 # BlockSearchExtension


## 功能概述

- **设置搜索关键词**：允许用户输入搜索关键词。
- **设置搜索模式**：用户可以选择搜索模式，包括“中文名”、“opcode”和“全部”。
- **搜索所有目标**：在所有角色中搜索匹配的积木块，并自动跳转到第一个结果。
- **搜索当前目标**：仅在当前编辑的角色中搜索匹配的积木块，并自动跳转到第一个结果。
- **跳转到下一个结果**：在搜索结果中跳转到下一个匹配项。
- **跳转到上一个结果**：在搜索结果中跳转到上一个匹配项。
- **跳转到特定索引**：根据用户输入的索引跳转到相应的匹配项。
- **清除高亮**：清除当前搜索结果的高亮标记。
- **获取结果数量**：返回当前搜索结果的总数。
- **获取当前结果索引**：返回当前高亮结果的索引。
- **获取当前结果信息**：返回当前高亮结果的详细信息。

## 安装与使用

1. **下载扩展**：从GitHub仓库下载 `BlockSearchExtension.js` 文件。
2. **加载扩展**：
   - 打开Scratch编辑器。
   - 点击右上角的“扩展”按钮。
   - 选择“从URL加载”。
   - 输入 `BlockSearchExtension.js` 文件的URL。
3. **使用扩展**：
   - 在积木面板中找到扩展图标。
   - 使用提供的积木块进行搜索和导航。



## 技术实现(部分失效)

- **积木定义**：使用Scratch的积木定义API来创建自定义积木。
- **搜索逻辑**：实现复杂的搜索匹配算法，支持关键词和模式匹配。
- **弹窗显示**：提供一个弹窗界面，显示所有搜索结果，并允许用户点击跳转。
